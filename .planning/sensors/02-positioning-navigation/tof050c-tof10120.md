## Overview

The TOF050C, TOF10120, and related mini ToF modules are small (~15×6 mm) pre-calibrated laser
distance sensors — essentially a VL53Lx die with an onboard MCU that speaks a simplified I²C or
UART protocol. Numeric suffix = max range in cm (TOF050C = 50 cm, TOF10120 = ~180 cm, TOF400C =
400 cm).

## Key specs (representative)

| Spec | TOF050C | TOF10120 | TOF400C |
|---|---|---|---|
| Range | 10–500 mm | 10–1800 mm | 10–4000 mm |
| Accuracy | ±5 mm | ±5% | ±1% |
| Interface | I²C / UART | I²C / UART | I²C / UART |
| Rate | 10–100 Hz | 10–100 Hz | 10–100 Hz |

## Interface & wiring

TOF10120 defaults to UART @ 115200 with a simple ASCII protocol, and also supports I²C (address
0x52). TOF050C is typically I²C-only at 0x52. Level-shifter generally not needed — most boards
have 3.3 V-compatible pullups.

## Benefits

- Tiny PCB — fits inside finger-sized enclosures.
- Pre-calibrated, no library needed for basic use.
- Dual I²C / UART on TOF10120 is flexible.

## Limitations / gotchas

- Vendor documentation is thin / Chinese-only / inconsistent.
- Default I²C address is the same across units — mux or XSHUT logic required for multiples.
- Behind-the-scenes this is a VL53 die — same sun / matte-black limitations apply.
- Firmware differs between batches; exact command set may shift.

## Typical use cases

- Handheld gadgets (distance-aware buttons, rangefinders).
- Tight robots where even a VL53 breakout is too big.
- DIY 3D scanners with a rotating platform.
